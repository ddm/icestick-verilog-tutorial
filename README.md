# iCEstick Verilog Tutorial

## Simulation

    > simulate.sh

Run `test_bench.v` against `rotate_leds.v` using [Icarus Verilog](http://iverilog.icarus.com/).

`CTRL-C` interrupts the simulation. Type `finish` to exit `iverilog`.
